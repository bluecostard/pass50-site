# PASS50 — Administration & Data V2

Déposer tous les fichiers de ce pack à la racine du dépôt GitHub en conservant les dossiers.

Après déploiement :
1. Ouvrir Administration → Data Hub → Synchroniser les profils.
2. Ouvrir Administration → Réseaux sociaux pour gérer Facebook, YouTube et les autres plateformes.
3. Ouvrir Administration → Actualité pour rechercher ou valider manuellement le lien original d’un buzz.
4. Vider le cache avec Cmd/Ctrl + Shift + R si nécessaire.
