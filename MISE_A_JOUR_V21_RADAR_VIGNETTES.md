# PASS50 V21 — Radar LIVE + vignettes FI + diaporama

Cette version regroupe dans une seule archive :

- le Radar LIVE V1 (`api/live-status.php`) ;
- la détection automatique des directs YouTube par rotation ;
- la fusion avec les directs ajoutés manuellement ;
- le retrait d'un direct terminé lors du prochain contrôle de sa chaîne ;
- la même vignette dans le Top 5 et dans la fiche influenceur ;
- la photo validée du profil comme visuel de secours ;
- un bouton de contrôle manuel du radar dans Administration → LIVE ;
- les images du diaporama affichées à 80 % d'opacité, soit environ 20 % de transparence.

## Limites connues

Instagram, Facebook, TikTok et Snapchat bloquent souvent la lecture publique automatisée de l'état LIVE. Ils restent en mode hybride pour cette première version. YouTube est la plateforme automatiquement contrôlée.

## Installation

Remplacer les fichiers de l'archive, y compris le dossier `api`, puis effectuer un rechargement forcé du navigateur.
