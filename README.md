# PASS50 — IONOS V1

Version fonctionnelle destinée à un hébergement Web IONOS classique.

## Inclus

- site public mobile-first ;
- inscription et connexion ;
- confirmation d’e-mail avec Brevo ;
- base MySQL/MariaDB ;
- 3 favoris et 5 suivis ;
- préférences de notifications ;
- duel « Qui est le plus coulé ? » avec résultats en pourcentage ;
- gestionnaire de photos et couvertures ;
- synchronisation du classement et de l’administration ;
- premier compte propriétaire.

Commencer avec `INSTALLATION_SIMPLE.md` ou ouvrir `install.php` après téléversement.


Le logo PASS50 noir, blanc et vert électrique est intégré au site et aux e-mails.
