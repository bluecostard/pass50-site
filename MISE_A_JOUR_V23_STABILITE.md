# PASS50 V23 — Stabilisation

Cette version consolide les menus administrateur pour éviter les écrasements entre modules, rend Contrôle qualité toujours accessible, renomme Données en Maintenance, clarifie son usage et place les flèches du Top 10 au-dessus des photos.

## Principes
- aucune donnée métier réinitialisée ;
- aucune modification des scores ;
- aucune suppression des liens, photos, naissances ou actualités ;
- cache renouvelé pour forcer le chargement des fichiers consolidés.
